## Subsidized Feeding Program Analytics
## Project Overview
This project demonstrates an end-to-end data analytics workflow applied to an operational field dataset tracking data collection and performance metrics for a regional subsidized feeding initiative in Kenya.

## Disclaimer
The dataset used in this project was shared for personal skills development 
purposes only. The organisation name has been removed to protect confidentiality. 
No proprietary or sensitive information is disclosed. This project is intended 
solely to demonstrate data analysis competency and does not represent or speak 
for any organisation.

## What I did
- Received a raw KoboToolbox survey export of 589 field staff responses 
  across 12 Kenyan counties
- Conducted a full data quality audit documenting six data issues
- Cleaned the dataset: removed empty columns, extracted school names from Universally Unique Identifier
  (UUIDs), standardised qualitative responses, converted ratings to numeric 
  scores, and flagged follow-up cases
- Built pivot tables analysing ratings by county, data collection mode by 
  region, follow-up schools by county, and staff role performance
- Designed a professional management dashboard with KPI cards, charts, 
  and a colour-coded follow-up table

## Tools used
Microsoft Excel — data cleaning, pivot tables, formulas, conditional 
formatting, dashboard design

## Key findings
- 97.8% of respondents rated the data collection process positively
- 13 schools flagged for follow-up based on Poor or Very Poor ratings
- Nairobi County accounted for 6 of the 13 follow-up cases
- Western Kakamega recorded the highest offline app usage, reflecting 
  adaptation to local connectivity conditions
- Kisumu and Mombasa recorded the highest average process scores (3.52/4.0)
